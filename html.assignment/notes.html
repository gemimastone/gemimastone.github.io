TAGS
< !DOCTYPE >
< html >
< body >
< title >
< head >
< !--...-- >
< blockquote >
< form >
DESCRIPTION
  • !DOCTYPE: This tag defines the document types
  • html: This tag defines an HTML document
  • body: This tag defines the document's body
  • title: This tag defines the given title to the document
  • head: This tag contains metadata/information for the document
  • !--...--: This tag is used to define a comment
  • blockquote: This tag defines a section that is quoted from another source
  • form: This tag defines an HTML form for users input
